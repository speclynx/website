What is ApiDOM?

ApiDOM is a semantic parser that provides a single, unifying structure for describing APIs across API description languages and serialization formats. Rather than requiring developers to handle each format individually, ApiDOM offers one consistent structure for all API specifications.

Like XML's DOM, ApiDOM provides query capabilities that decouple code from specific document structures, allowing semantic navigation independent of format variations.

Key Features

Semantic Parsing

ApiDOM operates through three processing stages - parse, refract, and generate - allowing it to transform various API specifications into a unified representation while preserving semantic meaning.

Format Unification

Support for multiple serialization formats (JSON, YAML) and API description languages, enabling cross-format conversions. Work with a single toolset regardless of the original document format.

Queryable & Traversable

Navigate API specifications semantically, independent of format variations. Query capabilities decouple your code from specific document structures.

Annotation Layer

Expand standard JSON structures by adding metadata (titles, descriptions) while maintaining semantic equivalence with the original data.

Installation

ApiDOM requires Node.js >=16.14.2 <=24. Install individual packages via npm:

npm install @speclynx/apidom-core

Or install specific adapters for your needs:

npm install @speclynx/apidom-parser-adapter-openapi-json-3-1
npm install @speclynx/apidom-parser-adapter-openapi-yaml-3-1
npm install @speclynx/apidom-parser-adapter-asyncapi-json-2
